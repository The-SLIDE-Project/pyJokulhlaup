# pyJökulhlaup README

## File contents

This repo contains everything necessary to run a suite of simulations in ISSM-GlaDS with a fully-coupled ice-marginal lake.

The experiments folder contains the data, parameterisation files, and analysis files necessary to run experiments.

Note, this does require an ISSM install with the necessary modifications to add ice-marginal lakes, which is avaliable at: 

Python dependancies are included in requirements.txt and we suggest installing a python3.12 virtual environment using venv or similar.
